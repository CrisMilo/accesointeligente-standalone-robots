## Herramientas usadas

* ORM: Hibernate 3
* ODBC: MySQL Connector 5.1.12
* Database Pooling: c3p0 0.9.1.2

## Configuración

Para hacer una instalación de esta plataforma, recomendamos:

Servidor Web: Apache Tomcat 6
Servidor de base de datos: MySQL
